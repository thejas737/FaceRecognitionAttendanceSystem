# Face Recognition Attendance System

A Python-based system that uses OpenCV to detect and recognize faces and automatically mark attendance.

## Features
- Real-time face detection
- Face recognition using LBPH
- Automatic attendance logging
- Simple and efficient implementation

## Technologies Used
- Python
- OpenCV
- NumPy

## How to Run
1. Capture faces using `capture_faces.py`
2. Train model using `train_model.py`
3. Run attendance system using `attendance_system.py`
